# Platformer

A 2D platformer game with Minecraft textures.

## Overview

This project is a 2D platformer game featuring Minecraft-style textures and classic platformer gameplay mechanics. Players navigate through levels, overcome obstacles, and explore an engaging game world.

## Tech Stack

- **Language:** C++
- **Build System:** CMake

## Features

- 2D platformer gameplay
- Minecraft-style textures and aesthetics
- Level-based progression
- Character movement and jumping mechanics
- Obstacle and platform navigation

## Prerequisites

- C++ compiler (GCC, Clang, MSVC, etc.)
- CMake 3.x or higher
- SFML or similar game library (if used)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/nikidastr3000/Platformer.git
   cd Platformer
   ```

2. Create a build directory and compile:
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

3. Run the game:
   ```bash
   ./Platformer
   ```

## Project Structure

- `src/`: Source code files
- `include/`: Header files
- `assets/`: Game textures and resources
- `CMakeLists.txt`: Build configuration

## Controls

- **Movement:** Arrow keys or WASD
- **Jump:** Spacebar
- **Pause:** ESC

## Gameplay

Navigate through platform levels, jump over obstacles, and reach the goal to advance to the next level.
